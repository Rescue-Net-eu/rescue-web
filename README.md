# Rescue-Net Website

This repository contains a minimal static website for the [Rescue-Net](https://github.com/Rescue-Net-eu/Rescue-Net.eu) project. The site fetches the backend health status from the Rescue-Net API and links directly to the main monorepo.

Open `index.html` in a browser while your backend is running to see the status message update.
